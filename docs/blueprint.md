# **App Name**: BKPM Q

## Core Features:

- Direct Kiosk Access: Kiosk registration page for direct access (no login required)
- Automated Numbering: Display automatically generated queue numbers.
- Real-time Status Display: Visually display the current status of the queue in real-time, suitable for TV monitor output
- Role Based Access: User roles to control app access and functions: Admin, Staff, Monitor TV, Registration Kiosk.
- Audible Queue Alerts: Audible announcement of queue number via text-to-speech for TV monitor.

## Style Guidelines:

- Primary color: Deep blue (#003049) to convey trust and professionalism.
- Background color: Light grayish-blue (#E6EEF2), desaturated to provide a calm and modern backdrop.
- Accent color: Soft orange (#D62828) for interactive elements and call-to-actions.
- Body and headline font: 'Inter', sans-serif for clear readability and a modern, neutral look.
- Use clean, minimalist icons to represent different services and actions.
- Implement subtle animations and transitions for user interactions to enhance engagement, using brief, simple fades or slides for queue updates.
- Design a layout optimized for large displays and smaller kiosk screens, using responsive design principles.
